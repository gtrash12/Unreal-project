# Unreal-project
언리얼엔진4로 개발한 3D 액션 MMORPG

# 조작법
- 이동 : WASD
- 공격 : 좌클릭
- 콤보공격 : 좌클릭 연타
- 찌르기 : 우클릭
- 대쉬공격 : 달리는 중 좌클릭 or 구르기 중 좌클릭
- 달리기 : Shift
- 구르기 : Spacebar
- 점프 : Shift + Spacebar
- 걷기 : ctrl
- 락온 타게팅 : 마우스 휠 버튼
- 락온 타게팅 오른쪽에 위치한 타겟으로 전환 : 마우스 휠 위로 회전
- 락온 타게팅 왼쪽에 위치한 타겟으로 전환 : 마우스 휠 아래로 회전
